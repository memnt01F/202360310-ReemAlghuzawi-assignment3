## Benefits & Challenges

### Benefits

Using AI provided several advantages:

Faster development process  
Clear explanations of programming concepts  
Help with debugging and fixing errors  
Improved code structure and organization  
Better understanding of JavaScript logic  
Support for writing documentation  
Reduced development time  
Improved problem-solving confidence  

AI also helped identify mistakes quickly and suggested solutions that improved code quality.

### Challenges

Some challenges were experienced when using AI:

AI responses sometimes required adjustments  
Generated code needed testing before use  
Understanding how the code works was necessary  
Not all suggestions matched the project requirements  
Careful review was needed to ensure correctness  

These challenges required critical thinking and manual verification of AI output.
